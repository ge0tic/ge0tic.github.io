--- title: Sobre el proyecto icon: fa-user order: 3 ---

ge0tic es 100% Software Libre, desde la plataforma base, hasta los módulos que lo componen. Entre las principales tecnologías que compenen ge0tic, están: OpenStreetMap (Mapnik) en OFFLINE, la gestión de información espacial utiliza el Gestor de Base de Datos PostgreSQL y su motor geoespacial PostGIS, los visores de mapas utilizan Librerías JavaScript para WebMapping OpenLayers y Leaflet; todas estas tecnologías fueron implementadas sobre una plataforma Linux Debian, que garantiza un mejor performance en comparación de otros sistemas operativos no libres.
